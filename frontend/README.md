# Frontend Service

Интерфейс приложения для просмотра фильмов и бронирования мест.

## Запуск

Сначала нужно запустить backend на `http://localhost:8080`.

Команды для frontend нужно выполнять из директории `frontend`.

```bash
python3 -m http.server 5173
```

Интерфейс будет доступен на `http://localhost:5173`.

## Основной функционал

- отображение списка фильмов
- отображение карточки фильма с картинкой
- просмотр мест в зале
- создание брони
- изменение выбранных мест в существующей брони
- отмена брони

## API

Frontend обращается к backend по адресу:

```text
http://localhost:8080
```

Используемые ручки:

- `GET /api/v1/movies` - получить список фильмов
- `POST /api/v1/booking` - создать бронь
- `PUT /api/v1/booking/{id}` - изменить бронь
- `DELETE /api/v1/booking/{id}` - отменить бронь

## Картинки фильмов

Карточка фильма берёт картинку из поля `image_url`.

Если `image_url` пустой, используется картинка по умолчанию:

```text
http://localhost:8080/static/images/cinema-night.svg
```

Статичные картинки хранятся в backend:

```text
backend/static/images
```

## Бронирования

Созданные через интерфейс брони сохраняются в `localStorage` браузера.

Это нужно потому, что в backend сейчас нет ручки для получения списка всех броней.
